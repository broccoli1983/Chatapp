# NobyAPIについて
## NobyAPIとは？
- 人工知能Noby(ノビィ)は会話型人工知能のWeb API。
- JavaScript等から簡単にアクセスでき、気軽にChatBotを実装する事が可能。
- Facebook、Twitter、Googleカレンダーなどのサービスとも連携。
- チャットでの会話を通じて、感情分析も可能。
- 参考サイト：https://webapi.cotogoto.ai/
